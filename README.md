# UX Research Copilot

Open-source AI assistant for UX researchers.

## Features

- Upload PDF reports
- Ask questions about research
- Answers with citations
- Local LLM (Ollama)
- RAG
- ChromaDB

## Tech Stack

- Python
- FastAPI
- Ollama
- ChromaDB
- LangChain
- Docker
